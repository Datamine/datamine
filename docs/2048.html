2048 for the Command-Line

It is likely that you've heard about Gabriele Cirulli's smash hit, the game 2048. Now it's available at your command-line, too! It's written in Python 2.7.8, and uses only standard libraries. It's developed for UNIX-based operating systems, though it should also work on Windows (untested). After downloading 2048.py, you can run it easily by typing python2 2048.py in your terminal.

To play, use your arrow keys to merge blocks of the same number. Try to get to 2048. On every turn, a new number (either 2 or 4) spawns. It is marked in bold. If you're a vim-user, then you can also use h, j, k, l instead of the arrow keys. The game tries to store your best score in a file in your home directory: ~/.2048.txt.

I considered making the game colorful, but decided against it, since the game's colorscheme might conflict with a user's terminal's colorscheme. If you want to play a really fancy, colorful version of command-line 2048, then I suggest checking out term2048.
